# Filetaro introduction website
If you are looking for Desktop app, click the [link](https://github.com/kkan0615/filetaro).

## Getting Started
### Install
```bash
yarn install
```
### Dev mode
```bash
yarn dev
```
### Build
```
yarn build
```
